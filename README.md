2017/9/5
	引用计数原则： 1.高层引用底层，底层不引用高层
				  2.同一层的，根据创建的逻辑顺序，先创建的引用后创建的，后创建的不引用先创建的
				  3.接口部分，Createxxx 类型的，需要客户端自己手动释放。GetXXX类型的，则不需要。Setxxx类型的，根据逻辑内部AddRef或者不加，但是要注意逻辑统一。
